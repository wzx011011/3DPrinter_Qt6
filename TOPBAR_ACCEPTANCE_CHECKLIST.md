# 顶部工具栏验收清单（CP7）

> 日期：2026-03-05  
> 目标：对 `main.qml` 顶栏全部可见入口做逐项功能验收，不漏项。

## 0. 预置条件

- 已启动应用并进入主界面。
- 至少准备一个可加载模型文件（建议 `.3mf` 或 `.stl`）。
- 当前页面可自由切换到「准备 / 预览 / 设备 / 在线模型」。

---

## 1. 文件入口（左上“三文件”）

| 编号 | 操作 | 预期结果 | 结果 |
|---|---|---|---|
| F-01 | 点击「三文件」展开菜单 | 文件菜单正常弹出 | ☐ |
| F-02 | 点「新建项目」 | 切到准备页；场景对象清空；项目名回到“未命名” | ☐ |
| F-03 | 点「打开项目...」并选可加载模型文件 | 模型开始加载；状态文案出现加载进度；完成后对象列表刷新 | ☐ |
| F-04 | 点「导入模型...」并选可加载模型文件 | 与打开行为一致，加载成功后留在准备页 | ☐ |
| F-05 | 点「保存项目」（已命名项目） | 不弹另存为，保存动作成功（最近文件保持最新） | ☐ |
| F-06 | 点「保存项目」（未命名项目） | 自动弹出“项目另存为”对话框 | ☐ |
| F-07 | 点「项目另存为...」并确认路径 | 项目路径更新为新路径 | ☐ |
| F-08 | 点「最近文件」子菜单任一条目 | 触发打开流程，成功后切到准备页并更新项目标题 | ☐ |
| F-09 | 点「清空最近文件」 | 最近文件列表为空，菜单项变灰 | ☐ |
| F-10 | 点「退出」 | 程序退出 | ☐ |

## 2. 快捷操作按钮（保存 / 撤销 / 重做）

| 编号 | 操作 | 预期结果 | 结果 |
|---|---|---|---|
| A-01 | 点击「保」按钮（已命名项目） | 直接保存，不弹窗 | ☐ |
| A-02 | 点击「保」按钮（未命名项目） | 自动弹“另存为” | ☐ |
| A-03 | 在准备页执行一次拖拽变换后点「↶」 | 变换被撤销 | ☐ |
| A-04 | 紧接上一步点「↷」 | 变换被重做 | ☐ |
| A-05 | 切到非准备页观察「↶/↷」 | 按钮禁用（置灰） | ☐ |
