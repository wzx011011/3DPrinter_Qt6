# Phase 81 Plan: Restore Preview Layer, Move, And Camera Controls

## Objective

Restore the screenshot-visible Preview layer rail, move/playback controls, and
camera fit interaction while preserving the existing Preview parser, GCV1
payload, and RHI renderer state.

## Tasks

1. Add RED tests:
   - `ViewModelSmokeTests` for `stepCurrentMove` clamping and G-code line
     synchronization.
   - `QmlUiAuditTests` for `PreviewLayerRail` wiring, move step controls,
     keyboard reuse of ViewModel stepping, and explicit Preview fit control.

2. Extend `PreviewViewModel`:
   - add `Q_INVOKABLE void stepCurrentMove(int delta)`,
   - route clamping through `setCurrentMove`,
   - keep atomic tool marker and G-code line window updates.

3. Add `PreviewLayerRail.qml`:
   - compact vertical `RangeSlider`,
   - top/bottom/current labels,
   - small step buttons,
   - direct calls into `setLayerRange`, `jumpToLayer`, and `moveLayerRange`.

4. Update `PreviewPage.qml`:
   - replace the simple vertical `Slider` with `Components.PreviewLayerRail`,
   - route keyboard left/right to `stepCurrentMove`,
   - add a fit button that calls `previewViewport.requestPreviewFit()`.

5. Update `MoveSlider.qml`:
   - add previous/next and jump-step buttons around the move slider,
   - keep existing slider and play/pause API,
   - call `stepCurrentMove` rather than doing arithmetic in QML.

6. Extend `RhiViewport`:
   - add `Q_INVOKABLE void requestPreviewFit()`,
   - reuse cached Preview fit bounds and avoid mutating `m_previewData`.

7. Verify:
   - focused CTest targets for `ViewModelSmokeTests` and `QmlUiAuditTests`
     after rebuilding affected tests if necessary,
   - `git diff --check`,
   - encoding guard.

## Acceptance Criteria

- Layer range/current-layer controls update `PreviewViewModel` and renderer
  bindings through real methods.
- Move playback buttons and keyboard shortcuts share one ViewModel stepping API.
- Preview camera fit/orbit/pan/zoom controls do not clear or repack
  `previewData`.
- Phase 81 verification status is `passed`.
