picked axis, and have the selected object rotate or scale on that axis with one
undo entry per drag.

  <action>
    Add QML-invokable rotate methods:
    `beginGizmoRotateDrag()`, `applyGizmoRotateDelta(int axis, float radians)`,
    and `endGizmoRotateDrag()`.

    Add QML-invokable scale methods:
    `beginGizmoScaleDrag()`, `applyGizmoScaleFactor(int axis, float factor)`,
    and `endGizmoScaleDrag()`.

    Begin records the selected source object and its starting rotation or
    scale. Apply validates the active source object, updates only the selected
    axis immediately, invalidates current-plate slice results, and emits
    `stateChanged()`. Rotate apply converts radians to degrees before adding
    to existing rotation. Scale apply clamps factor to at least `0.01` and
    multiplies only the picked axis. End pushes one `TransformCommand` from
    the recorded transform to the final transform, wrapping in a one-command
    macro when no macro is already open to avoid merging into a previous
    numeric transform.
  </action>

    - Multiple apply calls between begin/end create exactly one undo step.
    - Undo and redo restore the expected rotation/scale values.
