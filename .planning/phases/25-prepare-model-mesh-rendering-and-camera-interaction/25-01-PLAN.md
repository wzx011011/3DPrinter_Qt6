<threat_model>
| Threat | STRIDE | Mitigation | Disposition |
|--------|--------|------------|-------------|
| Malformed packed mesh bytes cause unbounded allocation | Denial of Service | Validate object counts, triangle counts, and remaining byte length before appending vertices | mitigate |
| Selection maps to wrong object because render IDs are reused as source indices | Tampering | Add explicit same-order source-index metadata and tests | mitigate |
| Inactive plate objects enter the QRhi model scene | Information Disclosure | Filter model batches by active source object indices in C++ scene data | mitigate |
| Selection-only state marks full model geometry dirty | Denial of Service | Separate mesh dirty state from selection/hover dirty state and test the transition | mitigate |
</threat_model>
