<threat_model>
| Threat | STRIDE | Mitigation | Disposition |
|--------|--------|------------|-------------|
| Renderer mutates editor selection from the render thread | Tampering | Emit picked source object from GUI-thread viewport and let `EditorViewModel` mutate selection | mitigate |
| QML implements object mapping logic | Tampering | Static audit forbids QML parsing/filtering and allows only signal forwarding | mitigate |
| Hover or selection forces full model buffer upload | Denial of Service | Separate selected/hovered state and audit upload helpers | mitigate |
| Right-click context menu is intercepted by picking | Reliability | Keep right-click outside QRhi camera/picking handling | mitigate |
</threat_model>
