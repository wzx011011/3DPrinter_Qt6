# Plan 75-01: Sidebar UI Restoration

## Goal

Make the Prepare left sidebar match the Phase 75 UI contract while preserving
the existing preset and scope behavior.

## Tasks

1. Backend/sidebar width
   - Update `BackendContext` sidebar min/default/max constants.
   - Update `PreparePage.qml` fallback width constants to match.
   - Keep existing persisted width clamp behavior.

2. Compact section support
   - Add a guarded compact property to `CollapsibleSection.qml`.
   - Apply compact mode only from `LeftSidebar.qml`.
   - Reduce section radius, header height, margins, divider, and content gap in
     compact mode.

3. Left sidebar density
   - Reduce `CxScrollView` margins and `ColumnLayout` spacing.
   - Apply compact mode to printer, filament, object, and object-settings
     sections.
   - Reduce process topbar/search/inline-panel spacing and radius.
   - Hide or keep hidden any visible placeholder controls.

4. Filament row restoration
   - Rework `FilamentSlot.qml` from a large color tile into a compact row.
   - Keep color swatch, 1-based slot number, preset combo, and compatibility
     state.
   - Preserve `requestCurrentFilamentPreset` behavior.

5. Process option row cleanup
   - Add QML display mapping for value-source keys in `OptionRow.qml`.
   - Add key-based label fallback for common Prepare option rows that remain
     English in upstream-schema mode.
   - Reduce compact row heights and compact field heights.
   - Keep SettingsDialog behavior compatible with `OptionRow` changes.

6. Verification and documentation
   - Run source audits for value-source raw keys and sidebar constants.
   - Run `git diff --check`.
   - Run encoding guard.
   - Record Phase 75 summary and verification docs.
   - Update ROADMAP/STATE completion markers if checks pass.

## Acceptance Criteria

- `SIDE-01`: sidebar width/density and printer/filament/process controls are
  visually closer to the target and preserve current interactions.
- `SIDE-02`: visible option labels/value-source text no longer expose raw
  internal source keys in the sidebar.
- `SIDE-03`: scope/search/settings/dirty affordances remain live and there are
  no new visible placeholders.
