### 30-04-T1 — Add PartPlate cache-invalidation unit test

**read_first:** `tests/PartPlateTests.cpp` (class shape from Phase 29); `src/core/model/PartPlate.h` (hasThumbnail, setThumbnail, addInstance, removeInstance, clearInstances, setLocked).

**action:** Add a private slot `void thumbnailCacheInvalidation()`:
1. Create a PartPlate, `setThumbnail(QImage(64,64,QImage::Format_RGBA8888))` (valid), `QVERIFY(plate.hasThumbnail())`.
2. `plate.addInstance(0,0)` → `QVERIFY(!plate.hasThumbnail())` (cache cleared).
3. Re-set, `plate.removeInstance(0,0)` → `QVERIFY(!plate.hasThumbnail())`.
4. Re-set, `plate.clearInstances()` → `QVERIFY(!plate.hasThumbnail())`.
5. Re-set, `plate.setLocked(true)` → `QVERIFY(!plate.hasThumbnail())`.

**acceptance_criteria:**
- PartPlateTests.cpp contains `void thumbnailCacheInvalidation()` slot
- 4 content-change operations each asserted to clear the cache
- test passes
