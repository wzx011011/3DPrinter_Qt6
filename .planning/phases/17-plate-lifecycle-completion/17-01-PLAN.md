    <action>
Add the plate lifecycle Q_INVOKABLE methods to `ProjectServiceMock` (service layer). In `ProjectServiceMock.h`, declare (next to the existing plate Q_INVOKABLE methods like togglePlateLocked, around line 96-104 of the .h):

Implement in `ProjectServiceMock.cpp`:

1. `clonePlate(sourceIndex)` (D-06): if `loading_` or source invalid, return false. Create a new plate via `m_plateList->createPlate()` (returns nullptr if MAX_PLATE_COUNT reached → return false). Copy metadata from source: name (append a " (副本)" suffix or " (copy)"), locked, bed type, print sequence, spiral, layer sequences, config (DynamicPrintConfig under HAS_LIBSLIC3R). Then deep-copy the objects: for each object index on the source plate (`m_plateList->objectIndicesOnPlate(sourceIndex)`), call the existing `duplicateObject(srcObjIdx)` machinery to create a real ModelObject copy in `model_->objects` + all parallel object arrays. `duplicateObject` already handles index shifting and appends the new object to the current plate — but here we need the new objects on the NEW (cloned) plate, not current. So: temporarily set current plate to the new plate, call duplicateObject per source object, capture the returned new indices, add them to the new plate's instance membership, then restore the previous current plate. Emit `projectChanged()` + `plateDataLoaded(count)`. Return true.

   NOTE: duplicateObject returns the new index and shifts higher indices, which can invalidate the source-plate object indices mid-loop. To handle this safely: first collect the source object indices into a list, sort descending, and duplicate in that order (so shifts don't affect unprocessed lower indices). This mirrors the deleteObject reverse-order pattern.

2. `movePlate(oldIndex, newIndex)`: if `loading_`, return false. Delegate to `m_plateList->movePlate(oldIndex, newIndex)`. Emit `projectChanged()` + `plateDataLoaded(count)` + `plateSelectionChanged()` (current may have moved). Return the delegate result.

3. `setPlatePrintable(plateIndex, printable)`: delegate to `m_plateList->setPlatePrintable(plateIndex, printable)`. Emit `projectChanged()`. Return true if plate valid.

4. `isPlatePrintable(plateIndex)`: `const PartPlate* p = m_plateList ? m_plateList->plate(plateIndex) : nullptr; return p ? p->isPrintable() : false;`
