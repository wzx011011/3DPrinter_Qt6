# Global UI Audit Fix Implementation Plan

**Goal:** Fix the directly actionable gaps from `.planning/audits/GLOBAL-UI-REVIEW.md` without starting large source-truth feature migrations.

**Scope:**
- Hide or classify top-level placeholder/no-op UI.
- Remove runtime phase/version copy from visible navigation and placeholders.
- Route feasible top-level actions to existing pages/dialogs instead of empty handlers.
- Convert the most visible chrome and sidebar values to `Theme.qml` tokens.
- Raise the most obvious sub-10px operational text in the affected sidebar controls.

**Out of Scope:**
- Full AssembleView implementation.
- Full ModelMall WebView implementation.
- Full calibration source-truth migration beyond currently wired pages/dialogs.
- Broad visual redesign not directly tied to the audit findings.

5. Write summary and commit only files touched by this quick task.
