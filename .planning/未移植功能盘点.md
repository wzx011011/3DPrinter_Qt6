## 按模块盘点的未移植功能

### 🔴 P0 — 核心功能缺失（影响主流程）

#### 1. 真机打印链路（Device/Cloud/Network 真实化）
- **上游**：bambu_networking（MQTT/lan 发现/打印发送）+ DeviceService + CloudService
- **我们**：5 个 Service 全 Mock（DeviceServiceMock/CloudServiceMock/CameraServiceMock/NetworkServiceMock/CalibrationServiceMock）
- **缺失**：
  - MQTT 协议接入（设备状态/控制）
  - SSDP/Bonjour 局域网设备发现
  - 真实打印发送（PrintDialog → SelectMachine → 发送到打印机）
  - 云端登录（OAuth + Creality/Bambu Cloud API）
  - 摄像头视频流（wxMediaCtrl → FFmpeg 解码）
- **工作量**：3000+ 行，需引入 MQTT/HTTP 库

#### 2. Calibration 真实化
- **上游**：Calib.cpp (15KB) + 9 种 CalibMode（FlowRate/PressureAdvance/TempTower/VolumetricRate/MaxVolumetricSpeed/RetractionTune/FlowRateProxy/Bucket/ManualLeveling）
- **我们**：CalibrationServiceMock 全合成，5 种占位
- **缺失**：真实切片校准 + 结果回写 preset + 4 种 CalibMode
- **工作量**：1500+ 行

#### 3. PartPlate 多板系统
- **上游**：PartPlate.cpp (258KB！) — 多板编辑/装配/拖拽/独立配置
- **我们**：PartPlateList 有基础渲染，但编辑/装配无
- **缺失**：多板独立切片配置 + 拖拽移动对象跨板 + AssembleView
- **工作量**：2500+ 行（258KB 上游）

### 🟠 P1 — 重要功能缺失（影响完整体验）

#### 4. Preset 管理完整链路
- **ProjectService 无 saveProject/exportProject**（v2.2 已知缺口）
- **PresetService 无 exportBundle**（v2.2 占位）
- **CreatePresetsDialog**（264KB，批量预设导入）

#### 5. 视频流 / 摄像头
- **上游**：wxMediaCtrl2.cpp (23KB) + MediaFilePanel.cpp (29KB)
- **我们**：CameraServiceMock 只有 URL setter，无解码
- **缺失**：实时视频流播放 + 延时摄影 + 媒体文件浏览/下载
- **依赖**：FFmpeg 或 Qt WebEngine

#### 6. AuxiliaryPage 接入
- **v2.2 已知**：9 卡片 handleCardAction 实现，但页面未挂载 main.qml
- **AuxiliaryService 语义错位**：UI 是分析工具，service 是文件复制

#### 7. Emboss 文字浮雕
- **上游**：GLGizmoEmboss.cpp (62KB) — 3D 文字添加/编辑
- **我们**：GLToolbars 有 Emboss 按钮 + EditorViewModel.addTextObject，但 gizmo 交互 UI 不全

### 🟡 P2 — 增强/打磨功能

#### 8. 未实现的上游 Gizmo（13/26）
- ✅ 已有按钮：Move/Rotate/Scale/Flatten/Cut/AdvCut/Support/Seam/Simplify/Measure/MeshBoolean/Emboss/SVG
- ❌ 缺交互 UI：Hollow/BrimEars/FuzzySkin/MmuSegmentation/FaceDetector/SlaSupports/Assembly
- 其中 MmuSegmentation（多色分区绘制）是多色打印核心

#### 9. i18n 翻译内容
- **qsTr 标记覆盖率**：~95%（良好）
- **翻译率**：en 16%（正常），zh_CN/ja/ko/de/fr **~0%**（全 unfinished）
- **缺失语种**：zh_TW/es/pt/ru/it/tr/uk（上游 13+，我们 6）

#### 10. ModelMall 模型商店
- **上游**：wxWebView 加载云端 mall URL
- **我们**：原生 QML mock 网格
- **依赖**：Qt WebEngine（vcpkg 未列）

#### 11. Home WebView / WebGuide
- **上游**：WebGuideDialog + DailyTips（MarkdownTip）
- **我们**：DailyTips 静态占位（PAGE-04），无 WebView

#### 12. KBShortcutsDialog
- **上游**：快捷键总览对话框
- **我们**：BBLTopbar 有 shortcutOverviewRequested 信号但无 dialog 接收

#### 13. 固件升级 / 插件管理
- **FirmwareDialog**：我们有占位
- **PluginManagerDialog**：我们有占位
- **UpgradePanel**：MonitorPage 占位
