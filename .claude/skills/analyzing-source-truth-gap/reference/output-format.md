# 分析输出格式

## 分析报告模板

```
## 差距分析: Px.x - 任务简述

| 项目 | 内容 |
|------|------|
| 任务 ID | Px.x |
| 上游模块 | `third_party/CrealityPrint/path/to/file.cpp` |
| Qt6 目标 | `src/core/...` 或 `src/qml_gui/pages/...` |
| 当前状态 | `[-]` / `[ ]` |

### 已实现的现有行为
- 具体已实现的功能点列表

### 缺失或不匹配的行为
- 具体差距条目（与上游一一对应）

### 建议的下一步根因迁移操作
- 具体操作建议（锚定上游行为）

### 实现后验证需求
- 需要运行时验证：是/否
- 验证命令：`scripts/auto_verify_with_vcvars.ps1`（如需要）
```

## 原则

- 以具体条目报告差距，而非模糊印象
- 每条差距必须关联到上游的具体文件/函数/行为
- 不提出自由设计的替代方案
